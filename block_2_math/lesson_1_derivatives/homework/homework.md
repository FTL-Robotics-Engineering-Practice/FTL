## Задача 1: Степенная функция для произвольного показателя

Докажите, что $(x^a)' = a \cdot x^{a-1}$ для **любого** вещественного $a$.

В классе мы доказали это только для натуральных $n$. Теперь нужно обобщить.

### Часть А: Отрицательные целые показатели

Докажите формулу для $x^{-n}$, где $n \in \mathbb{N}$.

**Подсказка:** Представьте $x^{-n} = \frac{1}{x^n}$ и используйте правило производной частного.

### Часть Б: Рациональные показатели

Докажите формулу для $x^{p/q}$, где $p, q \in \mathbb{Z}$, $q \neq 0$.

**Подсказка:** Пусть $y = x^{p/q}$. Тогда $y^q = x^p$. Продифференцируйте обе части по $x$, используя цепное правило для левой части (это называется неявное дифференцирование).

### Часть В: Произвольный вещественный показатель

Докажите формулу для $x^a$, где $a \in \mathbb{R}$, $x > 0$.

**Подсказка:** Используйте представление $x^a = e^{a \ln x}$ и цепное правило.

## Задача 4: Синус

Найдите производную функции $f(x) = \sin x$.

**Подсказка 1:** Используйте определение производной и формулу сложения: $\sin(x + h) = \sin x \cos h + \cos x \sin h$.

**Подсказка 2:** Вам понадобятся два замечательных предела:
- $\lim_{h \to 0} \frac{\sin h}{h} = 1$
- $\lim_{h \to 0} \frac{\cos h - 1}{h} = 0$

**Что должно получиться:** формула вида $(\sin x)' = ...$

## Задача 5: Косинус

Найдите производную функции $f(x) = \cos x$.

**Способ 1 (через определение):** Аналогично задаче 4, используйте формулу сложения: $\cos(x + h) = \cos x \cos h - \sin x \sin h$.

**Способ 2 (через синус):** Используйте тождество $\cos x = \sin\left(\frac{\pi}{2} - x\right)$ и цепное правило.

**Способ 3 (самый простой):** Используйте тождество $\sin^2 x + \cos^2 x = 1$. Продифференцируйте обе части!

**Что должно получиться:** формула вида $(\cos x)' = ...$

## Задача 6: Арксинус

Найдите производную функции $f(x) = \arcsin x$.

**Подсказка 2:** Из тождества $\sin^2 y + \cos^2 y = 1$ выразите $\cos^2 y$ через $\sin^2 y = x^2$.

**Подсказка 3:** Учтите, что для главного значения арксинуса $y \in [-\frac{\pi}{2}, \frac{\pi}{2}]$, поэтому $\cos y \geq 0$.

**Что должно получиться:** формула вида $(\arcsin x)' = ...$
