# Задание 1: Получение видимых клеток

## Описание

Нужно реализовать метод `get_visible_cells()` в классе `Sight`, который будет находить все клетки в секторе видения робота 270° (слепая зона сзади 90°).

## Что нужно сделать

Метод должен возвращать множество клеток (set), которые находятся:
- В радиусе `self.radius` клеток от робота (по умолчанию 4)
- В секторе 270° перед роботом (±135° от направления робота)
- Слепая зона 90° сзади (±45° от противоположного направления)

## Доступные переменные и параметры

### Параметры метода:
- `robot` - объект Robot с позицией и углом
- `grid` - объект Grid с информацией о сетке

### Поля объекта Sight (self):
- `self.radius` - радиус области видения в клетках (4)

### Поля объекта Robot:
- `robot.x` - позиция робота по X (в пикселях)
- `robot.y` - позиция робота по Y (в пикселях)
- `robot.angle` - угол робота (в радианах, 0 = вправо)

### Поля объекта Grid:
- `grid.cell_width` - ширина клетки
- `grid.cell_height` - высота клетки
- `grid.offset_x` - смещение сетки по X
- `grid.cols` - количество колонок
- `grid.rows` - количество строк

## Константа для использования

```python
max_angle_diff = math.radians(135)  # ±135° от направления робота = 270° сектор
```

## Алгоритм

1. **Определить область поиска:**
   - Вместо перебора всех клеток карты, определите квадратную область вокруг робота (от `robot_col - radius` до `robot_col + radius`).

2. **Перебрать клетки в этой области:**
   - Для каждой клетки `(col, row)` вычислить её центр в пиксельных координатах.
   
3. **Проверить расстояние:**
   - Вычислить расстояние от робота до центра клетки.
   - Если расстояние больше радиуса (или слишком маленькое, близкое к 0), пропустить клетку.

4. **Проверить угол (сектор видимости):**
   - Вычислить угол от робота к клетке, используя `math.atan2(dy, dx)`.
   - Найти разницу между этим углом и углом поворота робота.
   - **Важно:** Нормализовать разницу углов, чтобы она была в диапазоне `[-π, π]`.
   - Если модуль разницы углов меньше или равен `max_angle_diff`, значит клетка в зоне видимости.

5. **Вернуть результат:**
   - Собрать все подходящие клетки в множество (`set`) и вернуть его.

## Математические функции

- `math.sqrt()` - квадратный корень
- `math.atan2(y, x)` - угол (арктангенс) по координатам y и x
- `math.radians()` - перевод градусов в радианы
- `math.pi` - число π
