# Задание 2: Фильтрация препятствий

## Описание

Нужно реализовать часть метода `update()` в классе `Sight`, которая будет находить препятствия (черные клетки) среди видимых клеток.

## Что нужно сделать

В методе `update()` нужно:
1. Создать пустое множество `self.obstacle_cells`
2. Перебрать все клетки из `self.visible_cells` (которые вы нашли в предыдущем задании)
3. Для каждой клетки проверить, является ли она препятствием
4. Если да - добавить в `self.obstacle_cells`

## Доступные переменные

### Параметры метода update:
- `robot` - объект Robot
- `grid` - объект Grid

### Поля объекта Sight (self):
- `self.visible_cells` - множество всех видимых клеток (уже заполнено)
- `self.obstacle_cells` - множество клеток с препятствиями (нужно заполнить)

### Доступные методы:
- `grid.is_cell_filled(col, row)` - проверить, является ли клетка препятствием (возвращает True/False)

## Логика решения

1. Инициализируйте `self.obstacle_cells` как пустое множество (`set()`).
2. Пройдитесь циклом по всем координатам `(col, row)` в `self.visible_cells`.
3. Используя метод `grid.is_cell_filled`, проверьте каждую клетку.
4. Если клетка заполнена (есть препятствие), добавьте её координаты в `self.obstacle_cells`.

## Проверка

Запустите программу и нарисуйте препятствия в режиме карты (TAB). Переключитесь обратно в режим робота и убедитесь, что препятствия подсвечиваются другим цветом в зоне видимости.
