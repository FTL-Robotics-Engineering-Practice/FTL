# Задание 3: Вычисление центра масс

## Описание

Нужно реализовать метод `calculate_center_of_mass()` в классе `Sight`, который будет вычислять центр масс всех препятствий в зоне видимости.

## Что такое центр масс?

В данном случае центр масс - это просто **среднее арифметическое координат** всех точек препятствий. Представьте, что вы берете координаты всех препятствий и находите "среднюю" точку. Именно туда роботу нужно "смотреть", чтобы понимать, где находится основная масса препятствий.

## Что нужно сделать

Метод должен:
1. Проверить, есть ли препятствия в зоне видимости
2. Если нет - вернуть `None`
3. Если есть - вычислить среднюю позицию всех препятствий в пиксельных координатах
4. Вернуть кортеж `(x, y)` с координатами центра масс

## Доступные переменные

### Параметры метода:
- `grid` - объект Grid

### Поля объекта Sight (self):
- `self.obstacle_cells` - множество клеток с препятствиями

### Поля объекта Grid:
- `grid.cell_width`, `grid.cell_height` - размеры клеток
- `grid.offset_x` - смещение сетки

## Алгоритм

1. **Проверка на пустоту:**
   - Если список препятствий пуст, центр масс не определен. Верните `None`.

2. **Суммирование координат:**
   - Заведите переменные для суммы X и суммы Y.
   - Пройдитесь по всем клеткам с препятствиями.
   - Для каждой клетки вычислите её центр в **пикселях** (не в индексах колонок/строк!).
   - Прибавьте координаты к суммам.

3. **Вычисление среднего:**
   - Разделите накопленные суммы на количество препятствий.

4. **Результат:**
   - Верните полученные координаты (x, y).

## Проверка

Запустите программу, нарисуйте несколько препятствий и переключитесь в режим робота. В зоне видимости должна появиться красная точка - центр масс препятствий.
