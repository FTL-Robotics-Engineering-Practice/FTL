# Задание 2: Проверка коллизий

## Описание

Нужно реализовать метод `check_collision()` в классе `Robot`, который будет проверять, можно ли двигаться с заданной скоростью, или на пути есть препятствия.

## Что нужно сделать

Метод должен:
- Определить, в какую сторону движется робот (вперед или назад)
- Получить клетки соответствующего сектора (передний или задний)
- Проверить, есть ли в этом секторе препятствия
- Вернуть либо желаемую скорость (если путь свободен), либо 0 (если есть коллизия)

## Доступные переменные и параметры

### Параметры метода:
- `grid` - объект Grid с информацией о сетке
- `desired_linear_velocity` - желаемая линейная скорость (может быть положительной или отрицательной)

### Доступные методы:
- `self.get_sector_cells(grid, angle_offset)` - получить клетки в секторе
  - Используйте `angle_offset=0` для переднего сектора
  - Используйте `angle_offset=math.pi` для заднего сектора
- `grid.is_cell_filled(col, row)` - проверить, заполнена ли клетка (есть ли препятствие)

## Алгоритм решения

1. **Базовая проверка:**
   - Если желаемая скорость равна 0, верните 0.

2. **Определить сектор проверки:**
   - Если скорость положительная (движение вперед), нужно проверить передний сектор (`angle_offset=0`).
   - Если скорость отрицательная (движение назад), нужно проверить задний сектор (`angle_offset=math.pi`).
   - Получите список клеток для проверки, вызвав `self.get_sector_cells`.

3. **Проверка препятствий:**
   - Переберите полученные клетки.
   - Если хотя бы одна из клеток заполнена (используйте `grid.is_cell_filled`), значит есть препятствие.
   - Если найдено препятствие, верните скорость 0.

4. **Результат:**
   - Если препятствий не найдено, верните `desired_linear_velocity` без изменений.


Тест проверит, что метод правильно определяет коллизии в различных ситуациях.
