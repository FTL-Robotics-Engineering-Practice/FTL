# Задание: Система коллизий робота

## Описание

В этом задании вы реализуете базовую систему безопасности робота - обнаружение препятствий и предотвращение столкновений.

## Структура задания

Задание разбито на 3 последовательные части:

### 1. TASK_01_get_sector_cells.md
Реализация метода `get_sector_cells()` - получение клеток сетки в секторах движения (спереди и сзади).

**Файл:** `robot.py`
**Метод:** `get_sector_cells(grid, angle_offset)`

### 2. TASK_02_check_collision.md
Реализация логики остановки робота при обнаружении препятствия на пути.

**Файл:** `robot.py`
**Метод:** `check_collision(grid, desired_linear_velocity)`

### 3. TASK_03_draw_collision_zone.md
Визуализация зон проверки коллизий для отладки.

**Файл:** `renderer.py`
**Метод:** `draw_collision_zone(robot, grid)`

## Порядок выполнения

Выполняйте задания последовательно, начиная с TASK_01. Каждое следующее задание использует результаты предыдущих.

Для проверки первых двух заданий используйте специальные тесты:

```bash
python test_get_sector_cells.py
python test_check_collision.py
```

Для проверки третьего задания и финального результата запустите основную программу:

```bash
python code/main.py
```

## Управление

- **TAB** - переключение между режимом карты и режимом робота
- **W/S** - движение вперед/назад (режим робота)
- **A/D** - поворот влево/вправо (режим робота)
- **ЛКМ** - рисовать препятствия (режим карты)
- **ПКМ** - стирать препятствия (режим карты)
- **K** - сохранить карту
- **L** - загрузить карту
- **C** - очистить карту
- **Z** - отменить последнее действие

## Удобная разработка (Watcher)

Чтобы не перезапускать программу вручную каждый раз после внесения изменений в код, используйте скрипт `watcher.py`. Он автоматически перезапустит программу, когда вы закроете окно приложения.

Запуск через вотчер:

```bash
python code/watcher.py
```

Как это работает:
1. Вы запускаете вотчер.
2. Открывается окно программы.
3. Вы пишете код, сохраняете файл.
4. Нажимаете **ESC** в окне программы (или закрываете крестиком).
5. Вотчер тут же запускает программу заново с вашими изменениями.

## Результат

После выполнения всех заданий:
1. Робот **не должен** проходить сквозь стены и нарисованные препятствия.
2. Вокруг робота должны отображаться зоны проверки:
   - **Зеленые клетки:** путь свободен
   - **Красные клетки:** обнаружено препятствие

Эта система безопасности станет фундаментом для автономного движения в следующих шагах!
