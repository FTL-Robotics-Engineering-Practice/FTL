# Задание 1: Получение клеток в секторе

## Описание

Нужно реализовать метод `get_sector_cells()` в классе `Robot`, который будет находить все клетки сетки в секторе ±45° от заданного направления робота.

## Что нужно сделать

Метод должен возвращать множество клеток (set), которые находятся:
- В радиусе 3 клеток от робота
- В секторе ±45° от направления робота (всего 90° сектор)
- Учитывая `angle_offset` (0 для переднего сектора, π для заднего)

## Доступные переменные и параметры

### Параметры метода:
- `grid` - объект Grid с информацией о сетке
- `angle_offset` - смещение угла (0 для переда, math.pi для зада)

### Поля объекта Robot (self):
- `self.x` - текущая позиция робота по X (в пикселях)
- `self.y` - текущая позиция робота по Y (в пикселях)
- `self.angle` - текущий угол робота (в радианах, 0 = вправо)

### Поля объекта Grid:
- `grid.cell_width` - ширина одной клетки в пикселях
- `grid.cell_height` - высота одной клетки в пикселях
- `grid.offset_x` - смещение сетки по X (для боковой панели)
- `grid.cols` - количество колонок в сетке
- `grid.rows` - количество строк в сетке

## Константы для использования

```python
radius = 3  # радиус зоны обнаружения в клетках
sector_angle = math.radians(45)  # ±45° = 90° сектор
```

## Алгоритм решения

1. **Определить направление сектора:**
   - Вычислить угол направления проверки, сложив `self.angle` и `angle_offset`.

2. **Определить область поиска:**
   - Вместо проверки всей карты, ограничить область поиска квадратом вокруг робота (±radius).
   - Не забыть, что координаты в сетке и координаты робота связаны через `grid.offset_x`.

3. **Перебрать клетки в области:**
   - Для каждой клетки `(col, row)` вычислить её центр в пиксельных координатах.
   
4. **Проверить расстояние:**
   - Вычислить расстояние от робота до центра клетки.
   - Если расстояние больше радиуса (или слишком малое), пропустить клетку.

5. **Проверить угол (сектор):**
   - Вычислить угол от робота к клетке (`math.atan2`).
   - Найти разницу между углом к клетке и направлением сектора.
   - **Важно:** Нормализовать разницу углов в диапазон `[-π, π]`.
   - Если модуль разницы углов меньше или равен `sector_angle`, добавить клетку в результат.

6. **Вернуть множество найденных клеток.**

## Математические функции

Для решения понадобятся:
- `math.sqrt()` - квадратный корень
- `math.atan2(y, x)` - вычисление угла по координатам
- `math.pi` - число π
- `abs()` - модуль числа
